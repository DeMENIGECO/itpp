I++

Programmare in italiano.

I++ (Italian Plus Plus) è un linguaggio di programmazione interpretato dove puoi scrivere codice usando parole italiane. Semplice da imparare, potente da estendere.

Scarica I++ Documentazione

Caratteristiche

🇮🇹 Sintassi italiana

Scrivi programmi usando parole naturali: "stampa", "se", "ripeti", "definisci".

⚡ Interprete Python

Un interprete leggero scritto in Python, facile da modificare e migliorare.

🔄 REPL interattiva

Prova subito il linguaggio senza creare file.

🛠 Estendibile

Aggiungi nuove funzioni, comandi e librerie.

Esempio di codice

sia nome uguale a "Marco"

stampa "Ciao " + nome

se età è maggiore di 18 allora
    stampa "Adulto"
altrimenti
    stampa "Minore"
fine

Installazione

python -m itpp.main --repl
